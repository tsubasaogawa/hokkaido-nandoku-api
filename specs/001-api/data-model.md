# Data Model: 北海道の難読地名

## エンティティ

### `PlaceName`
北海道の難読地名とその読みがなを表すエンティティ。

| フィールド名 | 型 | 説明 | 制約 | 例 |
|---|---|---|---|---|
| `name` | string | 地名の漢字表記 | 必須, 空文字列不可 | "長万部" |
| `yomi` | string | 地名の読みがな（ひらがな） | 必須, 空文字列不可 | "おしゃまんべ" |

## データソース

- **形式**: CSV
- **ファイル名**: `data/nandoku_chimei.csv` (仮)
- **カラム**:
    1. `name` (漢字表記)
    2. `yomi` (ひらがな)
- **エンコーディング**: UTF-8
- **ヘッダー**: 1行目にヘッダー (`name,yomi`) を含む。

## 状態遷移
- このエンティティは静的なデータを表すため、状態遷移は存在しない。アプリケーション起動時にデータソースから読み込まれ、メモリ上で不変のデータとして扱われる。
